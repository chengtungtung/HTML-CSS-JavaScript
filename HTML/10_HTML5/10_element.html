重點筆記
  1. 快捷鍵
    • Shift + Alt + 向下鍵 => 複製後方下面
    • Alt + 向下鍵 => 可以移動單行
    • Shift + Alt + F => 整理格式
  2. id 的屬性值:(id的目的是識別)
    • 有區分大小寫
    • 至少包含一個字元
    • 不能以數字開頭
    • 不能包含空格
    • 建議不要有符號
  3. 不同元素可能有不同屬性：所以<h1>沒有href屬性，但<a>有href屬性
  4. 元素可以有多個屬性。
  5. 元素使用不支援的屬性，不會有效果：<h1>沒有href屬性
  6. on開頭的屬性就是跟js有關，是事件屬性。ex. onclick="js函式"
    • 要先宣告定義函式
    • function 函式名稱() { 要做的事情 }
